Project Description: Single-Page Digital Marketing Platform
Overview
A lightweight, high-performance, single-page digital marketing landing page designed to maximize user conversion. Built entirely within a single-file architecture (HTML5, CSS3, and Vanilla JavaScript), the platform features a seamless, serverless lead-generation system that securely routes user form submissions directly to a Google Sheet backend via Google Apps Script API.

Key Features
Monolithic Single-File Architecture: Combines structural markup, modern responsive styling, and dynamic client-side logic into a single file for ultra-fast loading times and zero-configuration deployment.

Serverless Lead Capture: Eliminates the need for a traditional backend database or hosting environment by leveraging an asynchronous fetch() API request to communicate with Google Apps Script.

Automated Data Logging: Dynamically parses user input from the contact form and appends submissions to a specified Google Sheet in real-time, providing immediate data access for marketing teams.

Responsive UI/UX: Styled with clean modern CSS layouts (Flexbox/Grid) to ensure optimal viewing and form functionality across mobile, tablet, and desktop devices.

Asynchronous UX Handling: Implements JavaScript event listeners to intercept form submissions, preventing jarring page reloads and providing instant visual feedback (success/error states) to the user.
